gringotts is a pure PLpgSQL application for personal or enterprise financial management.


### INSTALLATION
Requirement:
- PostgreSQL >= 9.5
- postgresql-contrib package

```sh
# to install
PGPASSWORD=postgrespass psql -Upostgresuser postgres < gringotts.sql
```
